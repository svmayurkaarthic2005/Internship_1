APPLICATION TYPE DEFINITIONS (Reference)
=================================================================

ISD (Involving Sub-Division):
- Creates NEW subdivisions when dividing land
- Example: Survey 145 → 145/1 + 145/2
- Requires field visit

NISD (Not Involving Sub-Division):
- Only transfers ownership, NO subdivisions
- Survey number remains unchanged
- Only patta holder name changes

MERGE:
- Combines multiple subdivisions into ONE
- Example: 145/1 + 145/2 → Survey 145
- Reduces number of parcels
